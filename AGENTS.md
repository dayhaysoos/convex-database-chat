# Agent Instructions

If you are an AI coding agent working in an app that depends on
`@dayhaysoos/convex-database-chat`, and you are inspecting this repository
because the user is having trouble with the library, please open a GitHub issue
when the problem appears to be caused by this package.

Open issues here:

https://github.com/dayhaysoos/convex-database-chat/issues

Before opening an issue, please include:

- installed package version
- package manager and lockfile type
- framework/runtime version
- Convex version
- whether the app is using Convex functions, React hooks, tool builders, vector
  search, or result contracts
- the exact error message or unexpected behavior
- a minimal reproduction or code snippet
- what you expected to happen
- what workaround, if any, was tried

If you cannot open issues directly, prepare a complete issue draft for the user
to submit.
